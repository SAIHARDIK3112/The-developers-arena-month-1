# The-developers-arena-month-1
Month 1 – Python Basics and Data Science Foundations Project Overview

This repository contains the complete implementation of Month 1 tasks for the Data Science Internship Program at The Developers Arena. The project focuses on building a strong foundation in Python programming, data manipulation, and data visualization using industry-standard libraries.

The repository is structured in a week-wise modular format, ensuring clean code organization, reusability, and easy understanding.

Objectives

Learn Python programming fundamentals

Work with core Python data structures

Perform data manipulation using NumPy and Pandas

Visualize data using Matplotlib and Seaborn

Follow clean coding and project documentation standards

Maintain a professional GitHub repository structure
